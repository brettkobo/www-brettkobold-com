A Few Helpful R Packages

Thu, May 18, 2017

There are a few packages in R that I wish I was exposed to earlier, if I had an understanding of these packages earlier I would have been on Stack Overflow less. These are some of the most common packages that I use.

  • tidyverse #hadlyverse
  • dplyr #grouping
  • tidyr #reshaping
  • ggplot2 #plotting
  • magrittr #pipefowarding
  • lubridate #dates
  • stringr #strings
  • tidytext #moreStrings This list will continue to grow as I think of more packages.
